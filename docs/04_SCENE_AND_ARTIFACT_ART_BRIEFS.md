# Scene and artifact art briefs

Этот файл дополняет `03_TECHNICAL_STACK_AND_ART_PIPELINE.md`. Он фиксирует содержание кадров и планируемые отличия. Сначала создаётся master A, затем каждое отличие по одному вносится в B локальным edit/inpaint.

## Общий style suffix для каждой сцены

Добавлять в конец каждого master prompt:

```text
Premium semi-realistic 2D illustrated archival scene for a find-the-difference puzzle game, late-1970s northern scientific expedition, documentary believable layout, premium hidden-object art quality, subtle painterly texture, realistic materials, clear object silhouettes, controlled medium-high detail, fixed 16:10 composition, all important objects in focus, cold muted northern palette with warm practical light accents. No readable text, no logos, no brands, no watermark, no UI, no captions, no close-up faces, no shallow depth of field, no heavy fog, no extreme darkness, no neon, no horror, no fisheye distortion.
```

## Общий edit suffix для B

```text
Edit only the masked region. Preserve exact camera, composition, lighting, color grading, texture, sharpness, perspective and every unmasked object. Make only the specified change. Keep the result pixel-aligned outside the mask. Do not reinterpret neighboring objects and do not introduce text, logos, clutter or global lighting changes.
```

**Роль:** tutorial, 4 крупных отличия.

**Кадр:** купе или общий вагон ночного поезда; камера чуть выше уровня глаз; стол у окна; спальные полки; багаж; мягкий янтарный свет; холодная ночь за окном.

**Объекты A:**

- латунные настенные часы;
- зелёный ремень на чемодане;
- тубус с картой возле левой ножки стола;
- шерстяное пальто;
- лампа;
- небольшой термос;
- билет без читаемого текста;
- снежное окно.

**Отличия B:**

4. тубус перемещён к правой ножке стола.

**Master prompt core:**

```text
Interior of a late-1970s overnight train compartment traveling through a snowy northern landscape at night. A small table under the window, a tea glass in a metal holder, brass wall clock, luggage with a green strap, a rolled map tube near the left table leg, coat, thermos, warm reading lamp, orderly believable travel clutter, no people.
```

# Acceptance rules for generated art

Сцена принимается только если:

- её можно понять без текста;
- все игровые объекты читаются на mobile;
- камера и crop A/B идентичны;
- незаявленных изменений нет;
- отличия соответствуют списку;
- каждое отличие можно описать словами;
- нет логотипов и мусорной типографики;
- нет мелких сломанных приборов;
- нет общей смены света между A/B;
- WebP не уничтожает мелкие детали.

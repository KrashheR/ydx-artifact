# Handoff: Главное меню выбора экспедиций — «Изумрудный меридиан»

## Overview
Экран выбора сюжетной кампании для приключенческой игры в жанре find-the-difference / archaeological mystery. Игрок выбирает одну из трёх экспедиций (каждая = карта из 13 уровней), видит свой прогресс, понимает условия разблокировки и может купить мгновенный доступ ко всем кампаниям через paywall-модалку.

Эстетика: экспедиционный архив конца 1970-х — тёмное дерево, латунь, патина, пергамент, топографические линии. Не free-to-play, не sci-fi, не неон.

## About the Design Files
Файлы в этом пакете — **дизайн-референсы, созданные в HTML**. Это прототипы, показывающие задуманный внешний вид и поведение, **а не production-код для прямого копирования**.

Задача — **воссоздать эти макеты в существующем окружении вашего проекта** (React / Vue / SwiftUI / Unity UI / native и т.д.), используя принятые в кодовой базе паттерны, компоненты и библиотеки. Если окружения ещё нет — выбрать подходящий стек и реализовать дизайн в нём.

Файл `Меню экспедиций.dc.html` — это «Design Component»: ему для рендера нужен рантайм `support.js` (тоже в пакете). Откройте HTML в браузере, чтобы увидеть все 6 фреймов на паннируемом холсте. Воспринимайте холст как Figma-страницу: 6 отдельных артбордов.

## Fidelity
**High-fidelity (hifi).** Финальные цвета, типографика, отступы и состояния. UI следует воссоздавать пиксель-в-пиксель средствами вашей кодовой базы. Атмосферные превью кампаний реализованы на CSS-градиентах/clip-path как плейсхолдеры — в проде их заменят на финальный арт (см. раздел Assets).

## Interactions & Behavior
- **Клик по доступной карточке** → плавный переход на карту кампании. Задумана анимация приближения (zoom-in) к превью, превью визуально трансформируется в карту экспедиции (~400–500ms, ease-out).
- **Клик по заблокированной карточке** → карточка слегка реагирует (shake/lift), показывается причина блокировки (tooltip/sheet); НЕ переход на карту. Предлагается пройти предыдущую кампанию или открыть все.
- **«Открыть все кампании»** (топ-бар или на карточке) → открытие paywall (desktop — центр-модалка, mobile — bottom sheet). Появление: fade + scale-in модалки / slide-up шита.
- **После покупки** → замки исчезают с короткой анимацией «снятия» (medallion fade/pop), пользователь остаётся на экране выбора, одна кампания мягко подсвечивается как рекомендуемая.
- **Hover доступной карточки (desktop):** `translateY(-3px)`, лёгкий zoom превью, латунная рамка ярче, мягкий внутренний ореол `0 0 26px rgba(184,138,69,.14)`, проступает намёк на маршрут (точки внизу превью). На mobile hover отсутствует — условия и статусы видны всегда.
- **Ambient-анимации (спокойные, не «казино»):** дрейф тумана/пыли в превью (`@keyframes fog`/`mist`, 18–24s, alternate); пульс активной точки маршрута (`pulse`, ~2.6s); шиммер скелетона загрузки (`shimmer`, 1.6s linear).

## State Management
- `campaigns[]`: `{ id, title, description, theme, levelsTotal:13, levelsDone, status, lastLevelName }`.
- `status` ∈ `'available' | 'in_progress' | 'completed' | 'locked'`.
- Правило разблокировки: кампания N+1 открыта, если кампания N `completed`, **или** куплен полный доступ.
- `hasFullAccess: boolean` — флаг покупки (хранится в профиле игрока; разблокирует все текущие и будущие кампании).
- `paywallOpen: boolean`, `paywallSource: 'topbar' | 'card'`.
- Состояния загрузки/ошибки превью — на уровне отдельной карточки (`previewState: 'loading' | 'ready' | 'error'`), карточка остаётся функциональной при ошибке изображения.
- `price` — приходит из стора платформы (App Store / Google Play / биллинг). В макете это переменная-плейсхолдер `{{price}}`, в проде подставлять локализованную цену.

## Assets
В макете изображений нет — все превью и иконки нарисованы кодом:
- **Превью кампаний** — CSS `radial/linear-gradient` + `clip-path: polygon(...)` для силуэтов гор/дюн/каноне + слой шума (inline SVG `feTurbulence`, `mix-blend-mode:overlay`). **В проде заменить на финальные атмосферные иллюстрации/рендеры** (рекомендуемый размер ≥ 2× контейнера превью; держать тёмный низ для читаемости бейджей). Контейнер уже даёт vignette `inset 0 -44px 60px rgba(13,20,26,.7)`.
- **Иконки** — инлайновые SVG (звук, настройки-слайдеры, замок, компас, ключ, галочка, лист, солнце, снежинка). Заменить на иконки из иконсета вашего проекта, сохранив штрих ~1.6–1.8px и цвета токенов.
- **Шум/грейн** — генеративный (`feTurbulence`), можно оставить или заменить текстурой бумаги.
- **Восковая печать-замок** — `radial-gradient` медальон + SVG замок; в проде можно заменить растровым ассетом печати.
Шрифты подключаются из Google Fonts; для оффлайна/прода — самостоятельно захостить `Cormorant Garamond` и `Manrope`.

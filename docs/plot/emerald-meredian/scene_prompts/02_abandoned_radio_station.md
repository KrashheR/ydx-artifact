# Emerald Meridian — Level 02

## Abandoned Radio Station / Заброшенная радиостанция

> **Критическое требование к генерации:** результат должен содержать **ровно 3 отдельные картинки в 3 отдельных файлах**. Нельзя объединять изображения в коллаж, сетку, триптих, split-screen или одно общее полотно. Каждая картинка должна быть самостоятельным файлом одинакового размера и формата 16:10.

## Готовый промпт

```text
ABSOLUTE OUTPUT REQUIREMENT — DO NOT IGNORE:
- Generate exactly 3 images as 3 separate image files.
- Image A, Image B and Image C must each be delivered as an individual standalone file.
- Do NOT combine the three images into one canvas.
- Do NOT create a collage, triptych, contact sheet, grid, split screen or side-by-side composition.
- Do NOT place multiple versions inside the same image.
- Each of the 3 files must have the same dimensions and a 16:10 aspect ratio.

Generate 3 separate images in one response for a find-the-difference game scene.

Required outputs:
1. Image A — the original unchanged expedition photograph.
2. Image B — the deliberately altered archive version with exactly 9 declared differences.
3. Image C — a solution/reference image based on Image A, with all 9 difference locations marked by thin clear circles.

CRITICAL OUTPUT RULES:
- create 3 separate image files;
- not a collage;
- not side by side;
- identical dimensions;
- aspect ratio 16:10;
- preserve exact camera, crop, lighting, perspective and all unchanged objects;
- Image C must match Image A except for solution circles;
- exactly 9 differences;
- no readable text, logos, brands, watermark, UI or captions;
- no visible people.

GLOBAL STYLE:
- premium semi-realistic illustrated hidden-object scene;
- late-1970s tropical expedition;
- humid jungle after a storm;
- grounded mystery;
- detailed and mobile-readable.

STORY CONTEXT:
The official report says the radio station was destroyed before the expedition disappeared. The original image proves that the equipment was still operational.

SCENE:
A small late-1970s communications hut surrounded by vines. A tall radio mast stands beside it. Through the open doorway, show a field radio, batteries, cables, a desk fan, weather instruments and a protected notebook. Fallen branches and wet foliage suggest a recent tropical storm.

KEY STORY CLUE:
The team transmitted messages after the supposed disaster.

THE EXACT 9 DIFFERENCES IN IMAGE B:
1. Break the upper section of the radio mast.
2. Remove the glowing indicator light from the radio set.
3. Disconnect the thick power cable from the battery.
4. Replace the intact window glass with a shattered pane.
5. Remove the dry notebook beneath the roof overhang.
6. Change the raised antenna switch to the lowered position.
7. Remove the spare radio tube from the desk.
8. Add a large fallen branch across the hut entrance.
9. Change the working desk fan into a rusted broken fan.

SOLUTION IMAGE:
Image C must reproduce Image A exactly and mark all 9 declared difference locations with thin clear circles.
```

# Песчаный меридиан — полный пакет для Codex

## Структура

- `prompts/` — сюжет кампании, промпт карты и 13 отдельных промптов игровых сцен.
- `map-handoff/` — координаты точек, маршрут, связь уровней с landmarks и готовый TypeScript-конфиг.

## С чего начать Codex

1. Прочитать `map-handoff/CODEX_TASK_PROMPT.md`.
2. Использовать `map-handoff/sand-meridian-map-layout.json` как главный источник правды.
3. Для TypeScript-проекта можно сразу импортировать данные из `map-handoff/sand-meridian-map.config.ts`.
4. Сопоставлять уровень и сцену через поле `promptFile`, которое указывает на файл в `prompts/`.
5. Общий сюжет находится в `prompts/00_campaign_story.md`.
6. Визуальный промпт фоновой карты находится в `prompts/00_map_prompt.md`.

## Ключевая гарантия

В пакете ровно 13 уровней и ровно 13 точек. Поля `order`, `id`, `x`, `y`, `promptFile`, `landmark` связывают код, сюжет и картинку карты в единую структуру.

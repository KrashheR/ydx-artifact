# Level 05 — Каменная арка / Rock Arch

## Сюжетный контекст
Под естественной каменной аркой маршрутные метки выстраиваются по полуденной линии, указывая на ось меридиана.

## Ключевая улика
Именно здесь игрок впервые понимает, что экспедиция шла не просто по местности, а по выверенной геометрии.

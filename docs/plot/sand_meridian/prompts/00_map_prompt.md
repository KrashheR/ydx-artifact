# Prompt — Campaign Map for “Sand Meridian”

```text
Create a stylized world-map / level-map illustration for a hidden-object mystery game campaign called “Sand Meridian”.

The map should feel like a warm desert counterpart to a northern expedition map: instead of icy coastlines and snow, show a vast sun-scorched desert region with dunes, rocky plateaus, canyon cuts, dry riverbeds, salt flats, an oasis, abandoned excavation zones, caravan traces, half-buried ruins, a weather station, and a remote desert research outpost.

Visual goal:
- premium illustrated game map;
- adventurous, mysterious, expedition vibe;
- warm, arid, sandy atmosphere;
- readable as a progression map for multiple levels;
- cinematic but clean;
- suitable as a campaign progression screen background.

Composition:
- top-down or slightly angled illustrated map view;
- the map should be moderately zoomed-in so the landmarks feel larger and clearer;
- show a clear journey path winding through the desert;
- include 13 progression points / stops connected by one continuous unbroken route from the first point to the last point;
- every point must be visually connected by this route with no gaps;
- the route should visually travel through varied landmarks:
  1) desert edge camp,
  2) caravan trail,
  3) dune basin,
  4) dry well,
  5) rock arch,
  6) canyon pass,
  7) abandoned dig site,
  8) oasis,
  9) salt flat,
  10) ruined observatory,
  11) buried temple courtyard,
  12) storm ridge,
  13) crater-like sunken basin / final secret zone.
- the path should feel like an expedition trail with discovery and escalation.

Style:
- semi-realistic illustrated map;
- hand-painted premium game-art look;
- high detail but still readable;
- warm palette: sand, ochre, amber, terracotta, dusty orange, sun-bleached stone;
- subtle age and mystery, but not fantasy overload;
- no realistic satellite texture;
- no cartoon simplification.

Important requirements:
- no readable text labels;
- no logos;
- no brands;
- no watermark;
- no characters as the main focus;
- no cluttered interface;
- no random icons unrelated to the setting;
- make each point location visually distinct enough that it could inspire a separate “find the difference” scene.

Optional extra detail:
- some route points may have small visual cues like tents, vehicle tracks, crates, excavation tarps, solar panels, ropes, survey poles, stone fragments, and wind-worn equipment;
- the final area should feel especially mysterious and visually important.

Aspect ratio: 16:10.
```

# Level 07 — Заброшенный раскоп / Abandoned Dig Site

## Сюжетный контекст
На полузаброшенном раскопе остаются следы поспешной работы и фрагмент важной архитектурной детали.

## Ключевая улика
Команда уже понимала ценность находки и старалась свернуть точку так, чтобы она выглядела малозначительной.

## Готовый промпт
```text
Generate 3 separate images in one response for a find-the-difference game scene.

Required outputs:
1. Image A — the main unchanged gameplay image.
2. Image B — the altered version with exactly 9 declared differences.
3. Image C — a solution/reference image based on Image A, with all 9 difference locations marked by thin clear circles.

CRITICAL:
- The three outputs must be 3 separate image files, NOT combined into one collage, NOT placed side by side in one canvas.
- All three outputs must have the same dimensions.
- Aspect ratio for each image: 16:10.

Global visual requirements:
- premium semi-realistic illustrated hidden-object scene;
- warm desert expedition aesthetic;
- late-1970s / retro expedition mood adapted to a hot arid environment;
- all important objects in focus;
- no readable text;
- no logos;
- no brands;
- no watermark;
- no UI;
- no captions;
- no random extra differences;
- no exaggerated fantasy magic effects;
- no unreadable mess.

Camera and composition:
- fixed camera angle across all outputs;
- medium-wide gameplay composition;
- all major props clearly visible;
- strong visual readability for gameplay;
- preserve the same crop, perspective, lighting direction, shadows, and object layout between Image A and Image B except for the exact declared differences.

Critical consistency rules:
- Image B must preserve the exact composition, camera, crop, lighting, perspective, textures, and all unchanged objects from Image A.
- Only the 9 declared differences may change.
- Do not introduce any additional accidental differences.
- Image C must match Image A and only add thin visible circles marking the difference locations.
- Do not alter the scene in Image C except for the circles.

Requirements for gameplay clarity:
- differences must be clearly visible but naturally integrated;
- differences should be distributed across the full image area;
- avoid clustering all differences in one place;
- avoid tiny impossible-to-see differences;
- avoid huge differences that break believability.

Style:
- polished game illustration;
- semi-realistic;
- rich texture;
- warm color palette: sand, amber, ochre, terracotta, dusty brown, faded red cloth, pale stone;
- inviting but mysterious;
- high detail suitable for a hidden-object / spot-the-difference game.

Output exactly 3 separate images:
- Image A: original scene;
- Image B: altered scene with exactly the 9 listed differences;
- Image C: Image A with all 9 difference locations circled clearly using thin circles.

Scene concept:
A premium semi-realistic illustrated hidden-object scene set at an abandoned archaeological dig site in the desert. Show torn excavation tarps, a shallow trench, a sieve frame, a tray with brushes, a ceramic jar, a lantern, a small tool kit, a rope coil, support planks, foreground footprints, and an exposed carved stone fragment or idol piece near the trench.

Use exactly these 9 differences in Image B:
1. The brush tray is moved from the sieve frame to the ground.
2. One ceramic jar is missing.
3. The hanging lantern is absent.
4. A torn tarp flap is tied up instead of hanging loose.
5. The trowel in the tool kit is replaced by a small hammer.
6. The rope coil becomes partially uncoiled.
7. The foreground footprint trail is removed.
8. A small excavation pennant changes color/tone.
9. The carved stone fragment is rotated noticeably.
```

# Задача для Codex — связать карту «Песчаного меридиана» с 13 уровнями

Используй файлы из этого архива как источник правды.

## Цель

Реализовать на карте кампании `sand-meridian` 13 интерактивных точек прохождения и соединяющий их маршрут. Каждая точка должна соответствовать конкретному сюжетному уровню и визуальному landmark на карте.

## Источники данных

1. `sand-meridian-map-layout.json` — координаты, порядок, IDs, landmarks и связь с prompt-файлами.
2. `sand-meridian-map.config.ts` — готовая TypeScript-версия данных.
3. `map-art-direction.md` — объяснение того, какой landmark должен быть под каждой точкой.

## Требования реализации

- Сохранять aspect ratio карты 16:10.
- Позиционировать точки по normalized координатам.
- Строить один непрерывный SVG path по `order`.
- Использовать стабильные IDs из конфига.
- Сделать состояния locked / available / current / completed.
- Добавить tooltip или карточку с `titleRu` и номером уровня.
- Клик по доступной точке должен открывать соответствующий уровень по `id`.
- Уровень 1 доступен сразу; каждый следующий открывается после завершения предыдущего.
- Не изменять coordinates и order без отдельного изменения data-файла.
- На desktop точки должны визуально лежать поверх landmarks.
- На mobile сохранить ту же карту и координаты; допускается pan/zoom.

## Проверка

После реализации убедись, что:

1. На карте ровно 13 точек.
2. Порядок идёт от 1 до 13 без пропусков.
3. Маршрут непрерывен.
4. Каждая точка соответствует своему landmark.
5. Финальная точка находится в скрытой впадине, а не у обсерватории или храма.
6. При resize точки не съезжают относительно фона.
